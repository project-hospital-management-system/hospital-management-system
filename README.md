## Quick Start

1. Extract ZIP to C:\\xampp\\htdocs\\
2. Start Apache & MySQL in XAMPP
3. Open phpMyAdmin: http://localhost/phpmyadmin
4. Create database: hospital_management
5. Import: config/schema.sql
6. Open: http://localhost/hospital_management_php/public/
7. Login: admin / admin123
